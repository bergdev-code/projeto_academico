# 🎓 Academic Management System
### *Um sistema completo para gerenciamento acadêmico, criado para fins educacionais.*

---

## 📌 Sobre o Projeto
O **Academic Management System** é um projeto acadêmico desenvolvido para simular um sistema real de gestão escolar.  
Ele permite administrar **alunos, cursos, disciplinas e notas**, oferecendo uma experiência prática para o aprendizado de programação, lógica e organização modular.

O sistema foi projetado para ser **simples**, **funcional** e **expansível**, servindo como base para estudos e melhorias futuras.

---

## 🚀 Funcionalidades Principais
- ✔️ Cadastro de **alunos**
- ✔️ Registro e gerenciamento de **cursos**
- ✔️ Sistema completo de **disciplinas**
- ✔️ Lançamento, edição e exclusão de **notas**
- ✔️ Exibição detalhada de informações acadêmicas
- ✔️ IDs automáticos e reorganização após exclusões
- ✔️ Estrutura totalmente modular em múltiplos arquivos
